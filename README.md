# Setup Instructions

Follow the steps below to get started with the project:

1. **Clone this Repository:**
   ```bash
   git clone https://github.com/avishshah11/brilliant-clone.git
   ```
2. **Install Dependencies**
   ```bash
   cd brilliant-clone
   npm install
   ```
3. **Run the Development Server**
   ```bash
   npm run dev
   ```
